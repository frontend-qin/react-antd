## Getting started

```bash
# clone the project
git clone https://github.com/frontend-qin/react-antd-admin.git

# enter the project directory
cd react-antd-admin

# install dependency
yarn install

# develop
yarn start
```

This will automatically open http://localhost:3030

## Build

```bash
# build for production environment
npm run build
```
